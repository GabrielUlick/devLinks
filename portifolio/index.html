MEU PORTIFOLIO

  • Listagem de Pokémon



    "Listagem de Pokémon" é um projeto interativo e responsivo que destaca minha paixão pelo universo Pokémon e minhas habilidades em tecnologias web. Desenvolvido com HTML, CSS e JavaScript, ele apresenta uma lista extensa de Pokémons, com nomes, imagens e descrições detalhadas. Com um design intuitivo e atrativo, os usuários podem alternar entre os modos claro e escuro, desfrutando de uma experiência visual personalizada. A aplicação responsiva se adapta perfeitamente a diferentes dispositivos, proporcionando uma experiência envolvente e educativa em qualquer tela. "Listagem de Pokémon" é uma jornada emocionante e interativa pelo mundo Pokémon, adequada para fãs e entusiastas de todas as idades.

  • CRUD Animal



    O CRUD Animal é uma aplicação móvel desenvolvida com React Native, TypeScript, Expo-SQLite e Expo, oferecendo funcionalidades de criação, listagem e exclusão de animais. Com interface intuitiva, proporciona uma experiência prática para gerenciar dados de animais. O repositório público está disponível para explorar o projeto e destacar as habilidades do desenvolvedor em tecnologias móveis modernas. Uma solução eficiente e funcional para o gerenciamento de informações.

  • Buscador API



    O Buscador API é um projeto desenvolvido com React e JavaScript, oferecendo aos usuários uma busca ágil e precisa por informações de CEPs através de uma interface intuitiva e responsiva. O objetivo é simplificar a experiência dos usuários ao encontrar detalhes de CEPs, tornando essa tarefa fácil e eficiente. A paixão pelo desenvolvimento web moderno impulsiona a criar soluções práticas e úteis.
