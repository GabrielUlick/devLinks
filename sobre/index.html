  • Sobre o Curso de Sistemas de Informação no IFES:

    O curso de Sistemas de Informação no IFES proporciona uma formação completa em tecnologia da informação, abrangendo disciplinas como programação, banco de dados, redes de computadores e inteligência artificial. Além das habilidades técnicas, o curso enfatiza competências como comunicação, trabalho em equipe e resolução de problemas. Estou confiante de que essa formação me tornará um profissional preparado para enfrentar os desafios do mercado de trabalho na área de tecnologia.


    Minha Experiência como Estagiário na SIAD Sistemas:

    Ao mesmo tempo em que estou focado nos estudos, também tenho a oportunidade de aplicar meus conhecimentos como estagiário Full Stack na SIAD Sistemas. A SIAD é uma empresa reconhecida por sua excelência em desenvolvimento de software e soluções inovadoras. Nessa experiência, tenho a chance de trabalhar com tecnologias de ponta, tais como:
    • JAVA uma linguagem versátil e amplamente utilizada no desenvolvimento de aplicações empresariais.
    • NODE.JS uma plataforma de código aberto que permite a construção de aplicações de rede escaláveis e de alta performance.
    • REACT uma biblioteca JavaScript para construção de interfaces de usuário interativas e eficientes.
    • ORM PRISMA uma ferramenta que facilita o acesso e manipulação de banco de dados através de um mapeamento objeto-relacional.
    • SQL a linguagem padrão para gerenciar e manipular bancos de dados relacionais.
    • MySQL e PostgreSQL dois dos sistemas de gerenciamento de banco de dados mais utilizados e poderosos.
